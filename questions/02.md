# Вопрос 2

Почему грамотный инженер добивается лучших результатов при работе с ИИ, чем обычный пользователь? Назовите как минимум три причины.

## Шкала оценивания

| Баллы | Описание |
|-------|----------|
| 0 | Ответ полностью неверный или студент пропустил вопрос. |
| 1–2 | Студент упоминает одну причину в общих словах, без конкретики. |
| 3–4 | Называет 1–2 причины, но объяснения поверхностные или с неточностями. |
| 5–6 | Перечислены 3 и более причин, но без раскрытия механизма каждой из них. |
| 7–8 | Названы не менее 3 причин с пояснением: системное мышление, умение формулировать задачи, критическое мышление, чувство архитектуры, управление техническим долгом. Ответ структурирован. |
| 9–10 | Максимально точный и полный ответ: каждая причина раскрыта с примером или объяснением механизма. Студент понимает, почему ИИ усиливает разницу между уровнями специалистов, а не сглаживает её. Структура чёткая, терминология корректная. |

## Критерии оценки

- **Полнота:** охвачено не менее 3 причин.
- **Глубина:** каждая причина объяснена, а не просто названа.
- **Точность терминов:** системное мышление, контекст, технический долг, архитектура.
- **Отсутствие ошибок:** не путает «что делает ИИ» и «что делает инженер».
